Implemented REST endpoints

The Web Model currently exposes six analytical views as JSON resources.

  • /OLAP/daily-sales-window
  • /OLAP/line-validation-ranking
  • /OLAP/operational-events-type
  • /OLAP/sales-ticket-area-cube
  • /OLAP/validations-network
  • /OLAP/telemetry-window

Architecture flow

SparkSQL analytical views are mapped as JPA read-only entities, queried through repositories and exposed through REST endpoints consumed by this dashboard.

SparkSQL View → JPA Entity → Repository → REST endpoint → Dashboard
